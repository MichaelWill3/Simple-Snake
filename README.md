## Snake

A very simple desktop Snake game built with Electron and TypeScript using HTML5 Canvas.

<img width="540" height="555" alt="image" src="https://github.com/user-attachments/assets/4647f0b1-9e7a-4559-9d5b-3c4675ab5ba0" />

### Controls
- Arrow keys / WASD to move
- P to pause, R or Enter to restart, M to mute
